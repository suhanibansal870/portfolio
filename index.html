A Machine Learning-Based Diagnostic Utility for Predicting Ertapenem Resistance in Klebsiella Pneumoniae Using MALDI-TOF MS Spectra

Summary: Built lightweight models using MALDI-TOF MS spectra to predict ertapenem resistance in Klebsiella pneumoniae quickly and accurately. Logistic Regression, MLP, and Random Forest achieved test accuracies of 86%, 84%, and 78%, supporting faster, targeted treatment decisions without slow or expensive genomic workflows.
